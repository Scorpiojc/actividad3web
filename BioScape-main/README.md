# BioLab Scape

Juego de plataformas 2D + shooter lateral desarrollado en Phaser.
## Controles

- A/D o flechas: mover
- Espacio: saltar
- Ctrl o click: disparar

## Objetivo

Escapar del laboratorio recogiendo las tres llaves: oro, plata y bronce.
La puerta final solo se abre cuando el jugador tiene las tres llaves.
